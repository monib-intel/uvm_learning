# UVM Learning

Learning and exploring Universal Verification Methodology (UVM) for digital design verification.

## Overview

This repository contains UVM-based testbenches and examples for studying advanced verification techniques.

## Contents

- Testbenches
- Verification IP (VIP)
- Example designs
- Test scenarios

## Getting Started

To use these UVM testbenches, you'll need:

- A SystemVerilog simulator (QuestaSim, VCS, Xcelium, etc.)
- UVM library installed and configured
- Basic understanding of SystemVerilog and UVM
